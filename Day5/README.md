# IT Help Desk Ticket Analysis Dashboard

## Overview

This project analyzes IT help desk ticket data to identify patterns in ticket volume, resolution time, common issues, and analyst performance. The goal is to derive actionable insights that improve operational efficiency and reduce recurring problems.


## Objectives

* Analyze ticket volume trends across time (day, month, hour)
* Evaluate resolution time performance
* Identify common issues and root causes
* Assess analyst performance
* Provide actionable recommendations

## Tools Used

* Microsoft Excel

  * Pivot Tables
  * Charts (Bar, Line, Column, Pie)
  * Slicers for interactivity


## Data Preparation

* Created calculated columns:

  * Resolution Time (in hours)
  * Day of Week
  * Month
  * Hour of Day
* Cleaned and structured the dataset for analysis

## Key Insights

### Ticket Trends

* Higher ticket volume observed during weekdays
* Peak activity occurs during specific hours of the day
* Certain months show increased workload

### Resolution Time

* Average resolution time varies based on category and priority
* High-priority tickets are resolved faster
* Complex issues take longer to resolve

### Top Problems

* Application crashes are the most frequent issue
* Network connectivity issues are common
* Hardware and account access issues also occur regularly

### Analyst Performance

* Significant variation in resolution time among analysts
* Top performers handle more tickets with lower resolution time
* Some analysts may require additional training

## Recommendations

* Implement automated password reset system
* Fix recurring application crash issues
* Improve network infrastructure
* Provide training for underperforming analysts
* Allocate resources during peak hours
* Create a knowledge base for common issues

## Dashboard Features

* KPI metrics (Total Tickets, Avg Resolution Time, Top Analyst)
* Interactive slicers (Analyst, Category, Priority)
* Visualizations for trends, performance, and issue distribution


## Conclusion

This analysis highlights key operational challenges in IT support systems. Addressing recurring issues and improving resource allocation can significantly enhance efficiency and reduce ticket volume.
