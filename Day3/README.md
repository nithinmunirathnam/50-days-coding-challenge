# 📊 Day 3 – Nutrition & Growth Analysis

## 📌 Project Overview

This project analyzes the relationship between children's dietary habits and their physical growth using a structured dataset. The goal is to identify key patterns between nutrition, lifestyle, and BMI to support better health decisions.


## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Validation
* Dashboard Design


## 📂 Dataset Details

The dataset includes the following attributes:

* Child ID
* Age (Months)
* Gender
* Height (cm)
* Weight (kg)
* Daily Fruit Servings
* Daily Vegetable Servings
* Daily Sugar Drinks
* Screen Time (Hours)
* Region

## ⚙️ Data Preparation

* Converted raw data into Excel Table
* Created calculated column:

  * **BMI (Body Mass Index)**
* Categorized BMI into:

  * Underweight
  * Healthy Weight
  * Overweight
  * Obese

## 📊 Analysis Performed

* Health Status distribution across age groups
* Impact of fruit and vegetable intake on BMI
* Relationship between sugar consumption and screen time
* Regional comparison of BMI and dietary habits


## 📈 Dashboard Features

* Interactive Dashboard created in Excel
* KPI Metrics:

  * Average BMI
  * Average Fruit Intake
  * Average Vegetable Intake
  * Average Sugar Consumption
  * Average Screen Time
* Slicers for:

  * Gender
  * Health Status
  * Region

## 🔍 Key Insights

* Higher sugar intake and screen time are linked to increased BMI
* Fruit and vegetable consumption improves health outcomes
* Lifestyle habits significantly impact child growth
* Regional differences exist in dietary patterns and BMI

## 💡 Recommendations

* Reduce sugary drink consumption through awareness programs
* Encourage daily fruit and vegetable intake
* Promote active lifestyle and reduced screen time
* Implement region-specific health strategies
