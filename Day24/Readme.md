# 🚗 Ride Booking Analysis (Day-24 - Python)

## 📌 Overview

This project is part of my **50 Days Coding Challenge**.
In this task, I performed basic data analysis on ride-booking data (similar to Ola/Uber) using Python.

Each ride record is stored as a tuple containing:

* Ride ID
* Driver Name
* City
* Fare
* Rating

## 📊 Dataset

```python
rides = (
    (101, "Ravi", "Bangalore", 250, 4.5),
    (102, "Anjali", "Hyderabad", 180, 4.2),
    (103, "Kiran", "Bangalore", 300, 4.8),
    (104, "Meena", "Chennai", 220, 4.0),
    (105, "Arjun", "Bangalore", 150, 3.9)
)
```

## 🎯 Tasks Performed

* ✅ Count total number of rides
* ✅ Extract rides from Bangalore
* ✅ Calculate total revenue from Bangalore
* ✅ Find highest rated driver
* ✅ Compute average rating per city
