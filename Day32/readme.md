# 🧑‍💻 Python Coding Challenge

## 📌 Overview
This project contains basic Python programs designed to improve logical thinking and understanding of control structures such as conditions and loops. It covers beginner-friendly problems that are essential for building a strong programming foundation.

---

## 🎯 Objective
To practice Python programming using if-else conditions, while loops, and problem-solving techniques.

---

## ❓ Questions

## 🔹 Level 1: Understanding Logic

### 1. Even or Odd Checker  
Write a program to check whether a number is even or odd using an if condition.  

### 2. Positive, Negative or Zero  
Write a program to classify a number as positive, negative, or zero using if-elif-else.  

### 3. Print Numbers from 1 to N  
Write a program to print numbers from 1 to N using a while loop.  

---

## 🔹 Level 2: Real Thinking

### 4. Sum of First N Numbers  
Write a program to calculate the sum of numbers from 1 to N using a while loop.  

### 5. Password Retry System  
Write a program that asks for a password and allows only 3 attempts before denying access.  

### 6. Multiplication Table  
Write a program to print the multiplication table of a number up to 10.  

---

## 🧾 Conclusion
This coding challenge helps in developing fundamental programming skills in Python. By solving these problems, learners gain a better understanding of conditions, loops, and logical thinking, which are essential for coding interviews and real-world applications.
