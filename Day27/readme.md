# Ride Booking Data Analysis (Using Tuples)

## Overview
This project simulates a real-world data analysis scenario for a ride-booking company like Ola/Uber. The dataset is stored using Python tuples, and various analytical tasks are performed to extract useful insights.

## Dataset
Each trip record contains:
(Trip_ID, City, Distance_km, Fare_amount)

## Tasks Performed

Task 1: Total Trips  
Find the total number of trips.

Task 2: Trips from Bangalore  
Count the number of trips that occurred in Bangalore.

Task 3: Total Revenue  
Calculate the total fare collected from all trips.

Task 4: High Distance Trips  
Display trips where distance is greater than 10 km.

Task 5: Highest Fare Trip  
Find the trip with the maximum fare amount.

Task 6: Average Fare (Bangalore)  
Calculate the average fare for trips in Bangalore.

## Technologies Used
Python

## Learning Outcomes
- Working with tuples for structured data  
- Applying filtering and aggregation logic  
- Using functions like len(), sum(), and max()  
- Performing basic data analysis operations  

## Conclusion
This project demonstrates how basic Python concepts can be used to perform real-world data analysis tasks effectively.
