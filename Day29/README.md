# 📊 Student Marks Analyzer

## 📌 Overview
The Student Marks Analyzer is a Python program that uses dictionaries to store student marks and perform basic analysis. It calculates total and average marks for each student, identifies the top scorer, and finds students who have failed in any subject.

## ❓ Questions
1. How to store student marks using dictionaries in Python?  
2. How to calculate total and average marks for each student?  
3. How to identify the top scorer among students?  
4. How to find students who failed (marks less than 40 in any subject)?  

## 🧾 Conclusion
This project demonstrates the use of Python dictionaries and basic programming concepts like loops and conditionals. It helps in understanding how to organize data, perform calculations, and extract meaningful insights such as top performers and failed students.
