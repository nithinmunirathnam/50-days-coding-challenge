# 🧑‍💻 Python Coding Challenge

## 📌 Overview
This project focuses on improving placement-level problem-solving skills using Python. It includes logical problems based on loops, conditions, and real-world scenarios such as data filtering and continuous data input systems.

---

## 🎯 Objective
To strengthen programming and analytical thinking skills required for placements by solving problems using loops, conditions, and basic data analysis techniques.

---

## ❓ Questions

## 🔹 Level 1: Placement Thinking

### 1. Number Guessing Game  
Store a secret number and ask the user to guess it.  
- If the guess is low → print "Too Low"  
- If the guess is high → print "Too High"  
- If correct → print "You Won!" and stop  

---

### 2. Reverse a Number  
Input: 1234  
Output: 4321  
Use a while loop to reverse the number.  

---

### 3. Count Digits in Number  
Input: 56789  
Output: 5  
Hint: divide the number repeatedly by 10.  

---

## 🔹 Level 2: Real-World Scenario (Data Analyst Thinking)

### 4. Sales Threshold Checker  
Given: sales = [1200, 800, 1500, 400, 2000]  
- Print sales greater than 1000  
- Count how many such values exist  

---

### 5. Continuous Data Entry System  
Keep asking the user to enter numbers:  
- Stop when the user enters 0  
- Print the total sum  

---

## 🧾 Conclusion
This challenge enhances logical thinking and introduces real-world problem-solving techniques. It helps in understanding loops, conditional statements, and basic data analysis concepts, which are essential for coding interviews and practical applications.
