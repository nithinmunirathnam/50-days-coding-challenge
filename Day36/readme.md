# 🧑‍💻 Student Batch Performance Analyzer

## 📌 Overview
This project focuses on analyzing student performance data using Python user-defined functions. It is designed as a practical data analyst challenge to improve programming, analytical thinking, and problem-solving skills.

The project uses lists, dictionaries, loops, conditions, functions, and aggregation techniques to generate meaningful insights from student batch data.

---

# 🎯 Objective
To build a Python-based student performance analysis system that can:
- Calculate average metrics
- Identify top-performing students
- Detect at-risk students
- Generate batch-wise summaries
- Compare batches and find the best-performing batch

This project improves logical thinking and introduces practical data analysis techniques using Python. It helps in understanding functions, loops, conditions, dictionaries, and aggregation methods that are commonly used in coding interviews, placement preparation, and real-world analytics applications.
