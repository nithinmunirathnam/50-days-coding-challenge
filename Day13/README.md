# 📊 Stress Level Analysis Dashboard

## 🧠 Overview
This project presents a Power BI dashboard that analyzes employee stress levels using key factors such as work hours, sleep duration, and caffeine intake. The dashboard helps identify patterns and provides insights to improve employee well-being and productivity.


## 🎯 Objectives
- Analyze employee stress levels  
- Identify correlation between work hours and stress  
- Evaluate impact of sleep on stress  
- Compare stress across departments  
- Provide actionable insights  


## 📌 Key KPIs
- Total Employees: 300  
- Avg Sleep Hours: 5.95 hrs  
- High Stress: 41%  
- Avg Work Hours: 8.59 hrs  
- Sleep-Work Ratio: 69.27%  


## 📊 Dashboard Features
- Interactive filters: Age Group, Department, Gender  
- Visualizations:
  - Avg Stress by Department (Bar Chart)  
  - Work Hours vs Stress Level (Scatter Plot)  
  - Work Hours by Stress Category (Line Chart)  
  - Stress Distribution (Donut Chart)  
  - Employee Data Table  


## 📈 Key Insights
- Employees working longer hours tend to have higher stress levels  
- Average sleep is below 6 hours, indicating imbalance  
- 41% of employees fall under high stress  
- Certain departments show higher stress compared to others  
- High stress is associated with increased caffeine intake  


## 💡 Recommendations
- Limit excessive work hours  
- Encourage better sleep habits  
- Conduct stress management programs  
- Focus on high-stress departments  
- Improve work-life balance initiatives  


## 🚀 Conclusion
The dashboard provides clear insights into how work patterns and lifestyle factors affect employee stress. These insights can help organizations take proactive steps to improve employee health and efficiency.
