# Insurance Claims Analysis (Using Lists)

## Overview
This project simulates a real-world insurance data analysis scenario using Python lists. It focuses on analyzing claim data and extracting useful insights for decision-making.

## Dataset
- policy_ids: Unique policy numbers  
- claim_amounts: Claim amounts in ₹  
- claim_status: Approved or Rejected  
- customer_age: Age of customers  

## Tasks Performed

Task 1: Total Approved Claim Amount  
Calculate total claim amount for approved claims.

Task 2: Count Rejected Claims  
Find number of rejected claims.

Task 3: High-Value Claims  
Display policy IDs where claim amount is greater than ₹10,000.

Task 4: Average Age of Approved Customers  
Calculate average age of customers with approved claims.

Task 5: Risk Category Tagging  
Categorize customers based on age:
- Age < 30 → Low Risk  
- Age 30–45 → Medium Risk  
- Age > 45 → High Risk  

## Technologies Used
Python

## Learning Outcomes
- Working with lists and indexing  
- Filtering and conditional logic  
- Data aggregation using sum() and count()  
- Basic data analysis techniques  

## Conclusion
This project demonstrates how Python can be used for basic insurance data analysis and risk categorization.
