# Coding Challenge Set: Strings & Tuples

## Overview
This project contains a set of beginner-friendly Python coding challenges focused on strings and tuples. These problems help improve fundamental programming skills such as string manipulation, iteration, and data handling.

## Challenges

Challenge 1: Find Longest Student Name  
Find the longest name from a given tuple of student names.

Challenge 2: Join Subjects into Single String  
Combine multiple subjects into a single string for display purposes.

Challenge 3: Reverse Each Student Name  
Reverse each student name in the tuple (useful for data masking).

Challenge 4: Count Vowels in Each Name  
Count the number of vowels present in each student name.

Challenge 5: Convert Tuple to Uppercase String  
Convert all course names into uppercase and display as a single string.

## Technologies Used
Python

## Learning Outcomes
- Understanding tuples in Python  
- String manipulation techniques  
- Using loops and comprehensions  
- Applying built-in functions like max(), join(), and sum()

## Conclusion
These challenges provide a strong foundation in Python basics and help build confidence in solving real-world data manipulation problems.
