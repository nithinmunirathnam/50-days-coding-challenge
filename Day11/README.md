# 🛡️ Insurance Claims & Fraud Analysis Dashboard

## 📌 Project Overview

This project presents an interactive **Insurance Claims & Fraud Analysis Dashboard** built using Power BI.
The goal is to analyze insurance claim data to identify trends, detect fraud patterns, and provide actionable insights for better decision-making.


## 🎯 Objectives

* Analyze claim trends over time
* Identify high-risk regions and incident types
* Detect fraud patterns and contributing factors
* Understand relationships between premium, deductible, and claim amount
* Provide business insights for risk reduction and cost optimization

## 📂 Dataset Description

The dataset contains insurance-related information such as:

* Policy details (policy number, state, premium, deductible)
* Customer demographics (age, occupation, education)
* Incident details (type, severity, location, date)
* Claim details (total claim amount, property, injury, vehicle claims)
* Fraud status (fraud reported: Yes/No)


## 🛠️ Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – KPI calculations


## 📊 Dashboard Features

### 🔢 Key Performance Indicators (KPIs)

* Total Claims
* Total Claim Amount
* Average Claim Amount
* Fraud Count
* Fraud Rate (%)
* Loss Ratio (%)

### 📈 Visualizations

* **Claims Trend Over Time** (Line Chart)
* **Claims Distribution by State** (Map)
* **Total Claim Amount by Incident Type** (Bar Chart)
* **Fraud vs Non-Fraud Analysis** (Bar Chart)
* **Premium vs Claim Amount** (Scatter Plot)


### 🎛️ Interactive Filters (Slicers)

* Time Period (Month/Year)
* State
* Incident Type
* Age Group


## 📌 Key Insights

* Collision incidents contribute the highest claim amounts
* Fraud cases account for approximately **25% of total claims**
* Certain states show higher claim frequency and fraud rates
* Weak correlation observed between premium and claim amount
* High-value claims are more likely to be associated with fraud

## 💡 Recommendations

* Implement stricter fraud detection for high-claim incidents
* Focus on high-risk regions for policy monitoring
* Introduce risk-based premium adjustments
* Improve claim verification processes for large claims
* Use predictive analytics for early fraud detection

## ⭐ Conclusion

This dashboard provides a comprehensive analysis of insurance claims and fraud patterns, enabling data-driven decision-making and improved risk management strategies.
