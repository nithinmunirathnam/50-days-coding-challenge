# 🧑‍💻 Smart Expense Analyzer

## 📌 Overview
The Smart Expense Analyzer is a Python program that uses basic operators to analyze monthly expenses and determine a user's financial status. It performs calculations for total expenses, savings, compares expenses, and evaluates financial health.

---

## 🎯 Objective
To use Python operators (arithmetic, comparison, logical, and assignment) to analyze financial data and make decisions based on savings.

---

## 📝 Problem Statement
Given the following data:

salary = 50000  
rent = 12000  
food = 8000  
transport = 3000  
entertainment = 4000  

---

## ❓ Tasks

### 🔹 Task 1: Total Expenses  
Use arithmetic operators to calculate total expenses.  

### 🔹 Task 2: Savings Calculation  
Calculate savings using:  
savings = salary - total_expenses  

### 🔹 Task 3: Expense Comparison  
Use comparison operators to check:  
- Is rent greater than food expense?  
- Is transport less than entertainment?  

### 🔹 Task 4: Financial Condition Check  
Use logical operators to determine:  
- If savings > 10000 → Good Saving  
- If savings between 5000 and 10000 → Average Saving  
- If savings < 5000 → Low Saving  

### 🔹 Task 5: Bonus Update  
Increase salary by 10% using assignment operator:  
salary += salary * 0.10  

---

## 🧾 Conclusion
This project demonstrates how Python operators can be applied to real-life financial analysis. It helps in understanding calculations, comparisons, logical decision-making, and updating values dynamically, which are essential skills in programming.
