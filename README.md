## 50 Days Data Analyst Challenge

This repository documents my **50 Days Data Analyst Challenge**, where I solve one real-world data analysis task every day to strengthen my analytical and technical skills.

## Objective
The goal of this challenge is to improve my practical knowledge in **data analysis, data visualization, and business insight generation** by working on daily challenges using industry tools.

## Tools & Technologies
 During this challenge, I will practice using:
  * Microsoft Excel
  * Power BI
  * SQL
  * Python
  * Data Visualization
  * Dashboard Development

## Repository Structure
 Each day's work is organized in a separate folder.
 Example:
  Day1 – Excel Data Analyst Challenge
  Day2 – Data Cleaning Task
  Day3 – Power BI Dashboard
  Day4 – SQL Data Analysis
  Day5 – Python Data Analysis

Each folder may contain:
 * Dataset used for analysis
 * Solution files (Excel / Power BI / SQL / Python)
 * Dashboard screenshots
 * README explaining the analysis and insights
