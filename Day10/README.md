# 📊 Instagram Engagement Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing Instagram post performance using Power BI. The goal is to identify key factors that influence engagement, such as post type, caption length, hashtags, and follower count.


## 🎯 Objectives

* Analyze engagement trends across different post types
* Identify top-performing content based on engagement rate
* Understand the impact of captions and hashtags on engagement
* Track reach and impressions over time
* Study the relationship between follower count and engagement


## 🛠️ Tools & Technologies

* Power BI
* Microsoft Excel
* DAX (Data Analysis Expressions)

## 📂 Dataset Features

* Post ID
* Post Date
* Post Type (Image, Video, Reel, Story)
* Caption Length
* Hashtags Used
* Likes, Comments, Shares, Saves
* Reach & Impressions
* Follower Count


## 📊 Dashboard Features

### 🔹 Trend Analysis

* Visualizes Reach, Impressions, and Engagement Rate over time

### 🔹 Top Performing Content

* Displays Top 5 posts based on engagement rate

### 🔹 Post Type Comparison

* Compares average engagement rate across different post types

### 🔹 Caption & Hashtag Analysis

* Shows impact of caption length and hashtags on engagement

### 🔹 Follower Growth Impact

* Analyzes relationship between follower count and engagement

### 🔹 Filters/Slicers

* Post Type
* Month Name

## 📈 Key Insights

* Videos have the highest engagement rate among all post types
* Stories show the lowest engagement performance
* Medium-length captions perform better than very short or long captions
* Using 10–20 hashtags improves engagement
* Engagement varies across months indicating seasonal trends
* Follower count does not always guarantee higher engagement


## 💡 Conclusion

Content type plays a major role in engagement, with video content performing the best. Optimizing captions and hashtag usage can significantly improve post performance.
