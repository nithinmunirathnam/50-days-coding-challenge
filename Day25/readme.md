# 📘 Coding Challenge Set: Strings & Tuples

## 📌 Overview

This repository contains a set of beginner-friendly Python coding challenges focused on **Strings and Tuples**. These problems are designed to improve fundamental programming skills, especially in **data manipulation, cleaning, and formatting**, which are commonly used in real-world data analysis and technical interviews.

The challenges cover:

* Formatting inconsistent data
* Counting and identifying unique values
* Cleaning unwanted spaces in strings
* Combining multiple data sources
* Extracting specific information from strings

---

## ❓ Questions

### 🔹 Challenge 1: Student Name Formatting

Convert all student names into proper case (first letter capital).

---

### 🔹 Challenge 2: Count Total Subjects

Find the total number of subjects and identify unique subjects from the given tuple.

---

### 🔹 Challenge 3: Remove Extra Spaces from Feedback

Clean the feedback strings by removing unwanted spaces.

---

### 🔹 Challenge 4: Combine Student Name & Course

Combine student names and their respective courses into a single tuple.

---

### 🔹 Challenge 5: Extract Email Domains

Extract domain names from the given email addresses.

---

## 🎯 Conclusion

These challenges help in building a strong foundation in **string operations and tuple handling** using Python. Mastering these concepts is essential for solving real-world problems in **data analysis, backend development, and coding interviews**. Practicing such problems regularly improves logical thinking and coding efficiency.
